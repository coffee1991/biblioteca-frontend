📚 Biblioteca Frontend

Aplicación web desarrollada en React que permite gestionar una colección de libros mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

🧠 Descripción

Este proyecto simula un sistema de gestión de biblioteca, donde los usuarios pueden:

Agregar nuevos libros
Visualizar la lista de libros
Editar información existente
Eliminar libros

Fue diseñado siguiendo una estructura modular y pensando en su integración futura con un backend en Python y una base de datos PostgreSQL.

🛠️ Tecnologías utilizadas
React
JavaScript (ES6+)
HTML5
CSS3
📦 Instalación y ejecución

Sigue estos pasos para ejecutar el proyecto en tu máquina local:

1. Clonar el repositorio
git clone https://github.com/coffee1991/biblioteca-frontend.git
2. Entrar al proyecto
cd biblioteca-frontend
3. Instalar dependencias
npm install
4. Ejecutar la aplicación
npm start

La aplicación se abrirá en:
👉 http://localhost:3000

🔄 Funcionalidades principales
📌 Creación de libros mediante formulario
📌 Validación básica de datos
📌 Edición de registros existentes
📌 Eliminación de libros
📌 Renderizado dinámico de la lista
🧩 Arquitectura

El proyecto está estructurado en componentes reutilizables, separando responsabilidades entre:

Vistas (pages)
Componentes
Lógica de estado

Está diseñado para integrarse fácilmente con una API REST.

🔗 Posible integración futura

Este frontend puede conectarse a un backend desarrollado en:

Python (Flask o FastAPI)
Base de datos PostgreSQL

Donde cada libro sería una entidad persistida en la base de datos.
